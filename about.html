About Smart Health Predictor System

The Smart Health Predictor System is a web-based application designed to help users monitor their health using intelligent analysis techniques.

This system calculates Body Mass Index (BMI) and provides health suggestions based on user inputs such as height, weight, and other health parameters.

The goal of this project is to create a simple, user-friendly platform that allows individuals to understand their health condition and take preventive measures early.

This project is developed using:

• HTML & CSS for Frontend Design
• Java for Backend Logic
• SQL for Database Management

The Smart Health Predictor System aims to combine technology and healthcare to provide better awareness and early prediction support. Created by student -- Jagriti Nishad
